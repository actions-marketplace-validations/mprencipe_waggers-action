# Waggers Swagger fuzzing

An action for fuzzing Swagger endpoints.

# Usage

See [action.yml](action.yml)

```yaml
steps:
- uses: mprencipe/waggers-action@SHA
  with:
    url: https://url/swagger-endpoint.json
```
